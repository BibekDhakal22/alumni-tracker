# 🎓 Alumni Tracker System

A web-based Alumni Tracking System built as a BCA Final Year Project, affiliated with Tribhuvan University (TU), Nepal.

## 🛠 Built With
- Python 3
- Flask (Web Framework)
- SQLite (Database)
- HTML, CSS, JavaScript
- Chart.js (Analytics)

## ✨ Features
- Student login with bcrypt password security
- Alumni profile management with photo upload
- Admin panel with search and filter
- Analytics dashboard with charts
- Alumni directory
- Notice board with pin feature
- Contact form and admin inbox
- Export alumni list to CSV
- Print / PDF report generation

## 🚀 How to Run
1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python app.py`
6. Open browser at: `http://127.0.0.1:5000`
